# Image generation

> **Status:** active (2026-07-11)

## Purpose

A briefed 2D image — UI icon, sprite, portrait, or texture — taken from prompt to a finalized file
in its target slot, with a review gate and a revise loop. Generation runs through the **Codex batch
contract** (`generation/BATCH.md`; `AGENTS.md` § Generation batches): Claude writes the batch and
reviews/files, the operator runs it in the Codex app, Codex generates into `generation/staging/`.
(Direct OpenAI / Figma one-offs remain an alternate path for trivial images.) Placement follows
**Pattern A** (symlink, zero-upload) per
[[0001-reference-the-shared-asset-library-via-external-locations]] — the brief's *target slot*
names the destination (library `ui/ | textures/` by default; a calling workflow may bind a
by-reference destination such as lore-brain `Media/visual/`).
