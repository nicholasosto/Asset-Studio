# 3D

> **Status:** experimental (2026-07-21)

## Purpose

The studio's 3D capability — props, rigs, and environment pieces blocked out in Blender and imported
into Studio. Process: [[3d-asset]].

## Standard

glTF source → `.rbxm` runtime. `[CAT]_[SUB]_[PascalName]_[Status]` with status `BLK → ALPHA → BETA → FNL`
per the TGL naming grammar (CAT ∈ PRP · RIG · EFX · ENV · LAY · ACC).

## Tooling

Blender MCP (the `blender-mcp-craft` skill) for modeling; Roblox Studio MCP for import;
`validate-3d-asset-art.mjs` for the source-art package.

## Output convention

Source packages in `models/<collection>/<slug>/` (the per-asset package — `README · manifest · boards ·
prompts/ · exports/`); runtime in `runtime/roblox/soul-steel/<cat>/<domain>/`. **Pattern B**
(Studio upload → `ServerStorage.TrembusGameLibrary`) per
[[0001-reference-the-shared-asset-library-via-external-locations]]; zones per
[[0008-restructure-the-shared-assets-library-into-staging-library-r]].

## Quality bar

Correct scale + origin; greyboxed blockout before detailing; passes `validate-3d-asset-art.mjs` and
`validate_asset_ids.py`.
